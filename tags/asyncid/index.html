Async Generators: yield Inside async def Functions Async generators answer the question, “What happens if you use yield inside a native async def coroutine function?” This concept might be confusing if you have some experience with using generators as if they were coroutines, such as with the Twisted framework, or the Tornado framework, or even with yield from in Python 3.4’s asyncio. Therefore, before we continue in this section, it is best if you can convince yourself that: Coroutines and generators are completely different concepts. Async generators behave much like ordinary generators. For iteration, you use async for for async generators, instead of ordinary for for ordinary generators.
